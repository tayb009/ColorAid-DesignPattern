# ColorAid Charts - Accessible Data Visualization

A comprehensive web application designed specifically for colorblind users, featuring accessible data visualizations with high contrast colors, pattern-based legends, and inclusive design principles.

## Features Implemented

1. Sidebar Navigation

Concept: A Navigation Pattern  where the menu is placed vertically on the side rather than the top.

2. Master–Detail

Concept: A pattern that shows a list + item details simultaneously. It allows users to browse a list of items and see the details of the selected item on the same screen.

3. Dashboards

Concept: A Data Display Pattern used to organize and present information clearly  in a single view.

4. Tooltips

Concept: An Interaction and Feedback Pattern used to show responses to user actions.

5. Miller’s Law

Concept: A principle stating that the average person can hold about seven objects in working memory.

6. Chunking

Concept: A Content Structuring Pattern used to help users comprehend content easily.
